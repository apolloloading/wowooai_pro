## 💡 三、人工测试侧重点提示

人工测试的价值在于**自动化测不到的部分**，请额外留意：

1. **视觉**：按钮对齐、字体大小、图标清晰度、颜色对比、暗色模式（如有）
2. **交互**：点击反馈延迟、Loading 状态合理性、错误提示是否人话
3. **文案**：错别字、不一致术语（"会话/对话"混用）、英文残留、专业术语解释
4. **键鼠操作**：Enter 发送、Shift+Enter 换行、Esc 关弹窗、Tab 顺序、复制粘贴
5. **异常路径**：断网时的提示、长时间没操作的状态、超长输入、特殊字符
6. **手感**：流式输出有没有抖动、滚动是否流畅、窗口缩放是否合理

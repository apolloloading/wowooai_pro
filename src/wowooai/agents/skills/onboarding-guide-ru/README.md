# 入职指引（人力窝 / 仁励窝）

人力窝（仁励窝）/ 久事附楼公司事务的官方知识库——员工日常工作中遇到的"WiFi 密码、VPN 怎么连、班车几点、报销怎么走、打印机怎么装"等问题，都能问。

## 适用场景

- 新员工入职第一天，办公环境/设备配置疑问
- 老员工偶尔需要查询某项制度（报销、6S、储物柜密码等）
- IT 配置（OpenVPN、WiFi、邮箱、共享盘、Cisco Jabber、会议室投屏）
- 通勤路线（9 号线小南门、南外滩金融直通车、接驳车）
- 行政后勤（快递收发、BFC 开心食堂、员工手册）

## 安装后怎么用

直接对 Agent 提问，例如：

- "公司 WiFi 密码是什么？"
- "OpenVPN 怎么配置？"
- "员工手册关于报销的规定？"
- "BFC 开心食堂怎么走？"

Agent 会自动检索本知识库并总结回答。

## 数据来源

阿里云百炼知识库（人力窝公司事务）。

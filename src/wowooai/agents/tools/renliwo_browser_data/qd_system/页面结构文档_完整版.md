## 三、通用操作模式

### Ant Design Select 下拉框

```python
async def ant_select(page, label, value):
    await page.locator('.ant-form-item').filter(has_text=label).locator('.ant-select-selector').first.click()
    await asyncio.sleep(0.4)
    await page.click('.ant-select-dropdown:not(.ant-select-dropdown-hidden) .ant-select-item-option:has-text("' + value + '")')
```

### 树形选择器（如福利所在地）

```python
# 福利所在地使用 Ant Design TreeSelect，含虚拟列表
# 需先点击打开，再滚动到目标项
async def tree_select(page, label, target_text):
    await page.locator('.ant-form-item').filter(has_text=label).locator('.ant-select-selector').first.click()
    await asyncio.sleep(0.5)
    # 可能需要滚动虚拟列表
    await page.evaluate(f'''() => {{
        const holder = document.querySelector('.ant-select-tree-list-holder');
        if (holder) {{ holder.style.overflowY = "auto"; holder.scrollTop = 150; }}
        setTimeout(() => {{
            const titles = document.querySelectorAll('.ant-select-tree-title');
            for (const t of titles) {{
                if (t.textContent === "{target_text}") {{ t.click(); return; }}
            }}
        }}, 500);
    }}''')
    await asyncio.sleep(1)
```

### 展开/收起筛选条件

多数页面默认**只显示部分筛选字段**（典型为 3 个），右侧"展开"按钮点击后显示全部筛选项。
例：增员派单页面默认可见 3 个筛选字段，展开后可见 27 个。

```python
# 展开按钮的实际结构: <a class="ant-pro-query-filter-collapse-button">展开</a>
expand_btn = await page.query_selector('a.ant-pro-query-filter-collapse-button')
if expand_btn:
    await expand_btn.click()
    await asyncio.sleep(1)
# 兼容写法:
# await page.click('text=展开')
```

### Tab 切换

```python
# 方法1：直接点击 tab 文字
await page.click('.ant-tabs-tab:has-text("后台处理中")')
await asyncio.sleep(3)

# 方法2（推荐，解决 Ant Design Tab 点击无反应问题）：
await page.evaluate('''() => {
    var tabs = document.querySelectorAll('.ant-tabs-tab');
    for (var i = 0; i < tabs.length; i++) {
        if (tabs[i].textContent.indexOf('后台处理中') !== -1) {
            tabs[i].dispatchEvent(new MouseEvent('click', {bubbles: true, cancelable: true, view: window}));
            break;
        }
    }
}''')
```

### 等待策略

| 场景 | 等待方式 |
|------|---------|
| 页面加载 | `wait_for_load_state('domcontentloaded')` + `sleep(3)` |
| 点击 Tab 后 | `sleep(3~8)` 等数据加载 |
| 展开筛选 | `sleep(1)` |
| 点击下拉框后 | `sleep(0.4~0.5)` |
| 查询后 | `sleep(3~5)` |
| 登录后 | `sleep(5)` |

**导出功能说明**:
- `导出参保信息`：直接下载 ZIP 包（含按城市分表 Excel 文件）；**无二次确认弹窗**；触发的 API：
  1. `POST /demo-qd-services/increase/exportIncreaseInsuredInfoCheck`（前置检查）
  2. `POST /demo-qd-services/increase/exportIncreaseInsuredInfo`（实际下载）
  - 下载文件名形如：`参保信息YYYYMMDDHHMMSS.zip`（ZIP 内 Excel 文件名通常为 GBK 编码，解压时需 `cp437→gbk`）
- `导出办理信息`：**弹出确认对话框**，含 2 个单选项：
  - "按办理事项导出"（默认选中）
  - "按险种导出"
  - 底部按钮：`Cancel` / `确认导出`
  - 点击「确认导出」后返回 blob 文件（需用 Hook + curl 方案下载）
- `下载参保信息空模板`：直接下载空模板文件
